# TangKang 医疗健康项目

GitHub 组织：https://github.com/tangkangyoudao

## 项目概述

TangKang 项目是一个面向糖尿病患者和医疗工作者的综合性健康管理平台，包含前端小程序、前端 Vue 管理后台和后端 Django 服务。

## 基本信息

- **项目名称**: TangKang（唐康）
- **项目类型**: 医疗健康全栈应用
- **当前状态**: ✅ 后端服务和后台管理服务已部署（2026-04-02）
- **部署方式**: Jenkins + Docker 自动化部署
- **云服务**: 百度云（已完成阿里云→百度云迁移）
- **创建时间**: 2026-04-01
- **GitHub 组织**: https://github.com/tangkangyoudao
- **最新分析**: 2026年1月-4月开发工作分析已完成
