## 一、系统概述

### 1.1 产品定位
唐康医疗健康小程序是一款面向糖尿病患者和医疗工作者的综合性健康管理平台，提供健康监测、医患沟通、健康指导等一站式服务。

### 1.2 核心价值
- **患者端**: 便捷的健康数据记录、智能分析、医患互动
- **医生端**: 患者管理、远程指导、健康干预
- **平台端**: 数据整合、智能提醒、健康知识传播

### 1.3 技术架构
- **前端**: 微信小程序 + Vant Weapp组件库
- **通信**: SSE (Server-Sent Events) 实时通信
- **图表**: ECharts 数据可视化
- **分包**: 主包+分包优化加载性能

## 三、核心功能模块详解

### 3.1 首页模块

#### 3.1.1 健康数据概览
- **血糖监测**: 实时显示最新血糖值，标注达标/过高/过低状态
- **数据图表**: 使用ECharts展示血糖变化趋势图
- **目标管理**: 显示个性化血糖目标范围
- **快捷入口**: 一键跳转到详细记录页面

#### 3.1.2 饮食管理
- **热量计算**: 实时计算今日摄入热量与剩余热量
- **目标设定**: 个性化每日热量摄入目标
- **记录提醒**: 未记录饮食时的提醒和快捷入口
- **健康食谱**: 接入智能食谱推荐系统

#### 3.1.3 功能快捷入口
```
1. 血压记录 - 血压监测与管理
2. 日常记录 - 综合健康日志
3. 用药记录 - 用药提醒与记录
4. 糖化血红蛋白 - 长期血糖控制指标
5. 尿微量白蛋白 - 肾脏功能监测
6. 足背动脉搏动 - 糖尿病足筛查
7. 手足麻木 - 神经病变监测
8. 视力模糊 - 眼部并发症监测
9. 饮食记录 - 详细饮食日志
10. 血尿酸记录 - 痛风风险监测
```

#### 3.1.4 医生关联
- **关联医师**: 快速查找和关联医生
- **健康档案**: 医生可查看患者完整健康档案
- **一站掌握**: 医生全面了解患者健康状况

### 3.2 记录模块

#### 3.2.1 饮食记录
- **食物录入**: 支持多种食物录入方式
- **热量计算**: 自动计算食物热量
- **时间记录**: 记录用餐时间
- **照片记录**: 支持拍照记录饮食

#### 3.2.2 日常记录
- **血糖记录**: 空腹、餐后等不同时段记录
- **血压记录**: 收缩压、舒张压、心率
- **运动记录**: 运动类型、时长、强度
- **症状记录**: 不适症状记录

#### 3.2.3 用药记录
- **药品管理**: 常用药品库
- **用药提醒**: 定时用药提醒
- **剂量记录**: 用药剂量和时间
- **效果反馈**: 用药效果记录

#### 3.2.4 专项检查记录
- **糖化血红蛋白**: 3个月平均血糖水平
- **尿微量白蛋白**: 早期肾病筛查
- **足背动脉搏动**: 血管功能评估
- **并发症监测**: 神经、眼部等并发症

## 五、业务流程说明

### 5.1 用户注册与登录流程
```
1. 微信授权登录
2. 手机号绑定（可选）
3. 个人信息完善
4. 健康目标设定
5. 医生关联（可选）
```

### 5.2 健康数据记录流程
```
1. 选择记录类型（血糖、血压、饮食等）
2. 输入数据值
3. 添加备注信息（可选）
4. 保存记录
5. 数据同步到云端
6. 生成分析报告
```

### 5.3 医患沟通流程
```
1. 患者发起咨询
2. 医生接收消息提醒
3. 医生查看患者健康数据
4. 医生提供专业建议
5. 患者接收并执行建议
6. 效果跟踪与反馈
```

### 5.4 健康干预流程
```
1. 系统识别健康风险
2. 自动生成健康提醒
3. 医生审核干预方案
4. 推送给患者
5. 患者执行干预措施
6. 效果评估与调整
```

**Q: 如何关联医生？**
A: 在首页点击"关联医师"，搜索医生姓名或医院，发送关联请求。

**Q: 如何查看健康趋势？**

**Q: 医生多久回复消息？**
