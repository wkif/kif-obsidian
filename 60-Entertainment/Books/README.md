## 📊 数据看板

### 本月阅读
```dataview
TABLE WITHOUT ID
  file.link as "书名",
  作者 as "作者",
  评分 as "评分",
  完成阅读 as "完成日期"
FROM "60-Entertainment/Books"
WHERE 完成阅读 >= date(today) - dur(30 days)
  AND file.name != "README"
SORT 完成阅读 DESC
```

### 按评分排序
```dataview
TABLE WITHOUT ID
  file.link as "书名",
  作者 as "作者",
  评分 as "评分",
  类型 as "类型"
FROM "60-Entertainment/Books"
WHERE 评分 AND file.name != "README"
SORT 评分 DESC
LIMIT 20
```

### 在读书籍
```dataview
TABLE WITHOUT ID
  file.link as "书名",
  作者 as "作者",
  开始阅读 as "开始日期",
  dateformat(date(today) - 开始阅读, "x") + " 天" as "阅读天数"
FROM "60-Entertainment/Books"
WHERE contains(状态, "在读")
  AND file.name != "README"
SORT 开始阅读 ASC
```

### 想读清单
```dataview
LIST
FROM "60-Entertainment/Books"
WHERE contains(状态, "想读")
  AND file.name != "README"
```

### 阅读统计
```dataview
TABLE length(rows) as "数量"
FROM "60-Entertainment/Books"
WHERE file.name != "README"
GROUP BY 类型
SORT length(rows) DESC
```

## 📖 阅读方法

### 快速阅读法
1. **浏览目录**：了解整体结构
2. **扫读章节**：找到关键信息
3. **精读重点**：深入理解核心内容
4. **复述总结**：用自己的话概括

### 主题阅读法
1. **确定主题**：明确想研究的主题
2. **选择书单**：找到相关的多本书籍
3. **对比阅读**：比较不同观点
4. **整合思考**：形成自己的理解

### 实践阅读法
1. **边读边记**：记录可实践的方法
2. **制定计划**：选择 1-3 个实践点
3. **执行验证**：在实际中应用
4. **总结反馈**：评估效果和调整
