# 🎬 Entertainment - 影视娱乐

> 记录看过的电影、剧集，以及观影感受和评价

## 📚 目录说明

### 🎥 Movies/（电影）
- 记录看过的和计划看的电影
- 评分、观影日期、观后感
- 按类型、年份、评分等维度管理

### 📺 TV-Shows/（剧集）
- 记录看过的和计划看的电视剧/网剧
- 追剧进度、评分、观剧感受
- 按类型、状态等维度管理

---

## 🎯 使用方式

### 状态标记

使用以下标签标记观看状态：
- `#想看` - 计划观看
- `#在看` - 正在观看
- `#看过` - 已完成观看
- `#弃剧` - 中途放弃

### 评分系统

使用 5 星评分：
- ⭐⭐⭐⭐⭐ 神作（5分）
- ⭐⭐⭐⭐ 优秀（4分）
- ⭐⭐⭐ 不错（3分）
- ⭐⭐ 一般（2分）
- ⭐ 较差（1分）

### 分类标签

**电影类型：**
- `#动作` `#喜剧` `#剧情` `#科幻` `#爱情`
- `#悬疑` `#惊悚` `#恐怖` `#动画` `#纪录片`

**剧集类型：**
- `#美剧` `#英剧` `#日剧` `#韩剧` `#国产剧`
- `#动漫` `#综艺` `#纪录片`

**主题标签：**
- `#推理` `#战争` `#历史` `#科技` `#犯罪`
- `#家庭` `#成长` `#职场` `#奇幻` `#超级英雄`

## 📊 观影统计

### 本月观影
```dataview
TABLE WITHOUT ID
  file.link as "作品",
  rating as "评分",
  观看日期 as "日期"
FROM "60-Entertainment"
WHERE 观看日期 >= date(today) - dur(30 days)
  AND file.name != "README"
SORT 观看日期 DESC
```

### 按评分排序
```dataview
TABLE WITHOUT ID
  file.link as "作品",
  rating as "评分",
  类型 as "类型"
FROM "60-Entertainment"
WHERE rating AND file.name != "README"
SORT rating DESC
```

### 想看清单
```dataview
LIST
FROM "60-Entertainment"
WHERE contains(tags, "#想看")
```

### 正在看
```dataview
TABLE WITHOUT ID
  file.link as "作品",
  进度 as "进度"
FROM "60-Entertainment"
WHERE contains(tags, "#在看")
```

### 统计面板
```dataview
TABLE length(rows) as "数量"
FROM "60-Entertainment"
WHERE file.name != "README"
GROUP BY contains(string(tags), "看过") as "已看",
         contains(string(tags), "在看") as "在看",
         contains(string(tags), "想看") as "想看"
```

## 💡 使用技巧

### 创建新记录
1. 在对应目录创建文件（电影在 Movies/，剧集在 TV-Shows/）
2. 使用模板快速创建（见下方模板）
3. 填写基本信息和观影感受

### 标签策略
- 使用状态标签追踪观看进度
- 使用类型标签方便检索
- 使用主题标签发现相似作品

### 链接关联
- 同类型作品互相链接
- 同导演/演员作品关联
- 相似主题作品推荐
