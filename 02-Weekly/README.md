# Weekly（周总结）

用于整理一周的收获。

这里是：

Daily → 深度思考

的重要桥梁。

---

## 什么时候写？

每周一次。

建议：

周日晚上。

---

## 写什么？

1. 本周完成了什么
2. 遇到的关键问题
3. 学到的最重要知识
4. 新产生的点子
5. 可以写成博客的主题
